# PDF Splitter

Ferramenta em Python para **dividir automaticamente um PDF em vários arquivos** com base em usuários ou setores. Ideal para separar documentos grandes (relatórios, holerites, comunicados) em arquivos menores, um por destinatário, sem trabalho manual.

## O problema que resolve

Separar manualmente um PDF de centenas de páginas por pessoa ou setor é demorado e sujeito a erro. Esta ferramenta automatiza essa divisão em segundos.

## Funcionalidades

- Divide um PDF em múltiplos arquivos automaticamente
- Separação por usuário ou setor
- Geração de arquivos menores e organizados por destinatário

## Tecnologias

- Python 3
- Bibliotecas de manipulação de PDF (ver `requirements.txt`)

## Como rodar

```bash
# Clone o repositório
git clone https://github.com/David-Henrique-Cangiani/pdf-splitter.git
cd pdf-splitter

# Instale as dependências
pip install -r requirements.txt

# Execute
python pdf-splitter.py
```
